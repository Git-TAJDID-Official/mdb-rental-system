# Project Overview

## Project Name
Majlis Daerah Besut Rental System

## Owner
TAJDID Corporation 

## Pengguna
1. Majlis Daerah Besut

## Objective
i.	Menyediakan platform digital bagi menggantikan proses penyewaan manual yang menggunakan borang kertas.
ii.	Memudahkan pengguna membuat permohonan sewa tanpa perlu hadir secara fizikal ke pejabat pengurusan.
iii.	Menyimpan semua maklumat penyewaan secara berpusat dan sistematik bagi mengelakkan kehilangan data.
iv.	Meningkatkan kecekapan proses permohonan, penyambungan dan penolakan sewa melalui sistem yang lebih pantas dan teratur.
v.	Menyediakan saluran aduan secara dalam talian bagi memudahkan pengguna melaporkan sebarang isu berkaitan penyewaan.
vi.	Membantu pihak pentadbir dalam memantau dan mengurus permohonan serta aduan dengan lebih efisien.


## Core Modules
- User Management
- Premis Listing Management
- Rental Application
- Continue Rent Application
- Approval/Reject Rental Workflow
- Payment Tracking
- Reporting Dashboard
- Feedback/Complaint
